# Application de Gestion de Photos

## Description
Cette application permet aux utilisateurs de gérer leurs photos :
- télécharger des images
- créer des albums
- générer des souvenirs vidéo
- supprimer et restaurer les images

## Technologies utilisées
- Python
- Django
- HTML
- CSS
- JavaScript

## Fonctionnalités
- Upload d'images
- Création d'albums
- Galerie photos
- Génération de souvenirs vidéo
- Corbeille (suppression après 7 jours)
